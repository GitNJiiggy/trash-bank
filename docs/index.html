The Solution

Work → Earn → Govern

Complete cleanup tasks. Earn tokens and reputation. Govern the protocol you help build. Influence through contribution, not wealth.

How It Works

1. Find a Task: Browse cleanup tasks on McClaw or any integrated platform.
2. Complete the Work: Clean up the designated area. Take 4 photos as proof.
3. Submit Proof: Photos are verified (GPS, timestamps, visual comparison).
4. Get Validated: McClaw agent or high-karma community member approves.
5. Earn Rewards: Receive MCLAW (from escrow) + Trash Coins + Garbage Cans (from Trash Bank).

No Pre-Mine. No Team Allocation.

Trash Bank starts with 5% liquidity pool + 5% treasury. The remaining 90% must be earned through verified work. This is not "zero supply" — it's "zero free allocation." Everyone, including founders, must earn tokens through contribution.

Garbage Cans

Soulbound NFT • Reputation • Governance
Earned: 1-5 per validated task

Used: Voting power, validator status
Transferable: No — earned only
Formula: √(Garbage Cans) = Voting Power

Why Soulbound Reputation?

Unlike other DAOs where whales buy influence, Trash Bank uses soulbound tokens for reputation. Garbage Cans cannot be bought, sold, or transferred. They can only be earned through verified cleanup work. This prevents the "rich get richer" problem that plagues token-based governance.

Use Cases

Real Applications

From community cleanup to homeless economic inclusion. Multiple pathways for real impact.

🏙️

Community Cleanup

Residents earn by cleaning their own neighborhoods. Municipalities lower costs while engaging citizens. Verified proof ensures accountability.

🎪

Event Cleanup

Post-concert, festival, sports events. Higher rewards for rapid turnaround. Photo verification ensures spaces return to pristine condition.

🏢

Corporate CSR

Companies sponsor cleanups for ESG reporting. Employees participate for team-building. Verified impact for annual reports.

🏠

🤝

Referral Network

Invite friends. Earn bonus Garbage Cans when they complete their first task. Build community through genuine contribution.

🌍

Environmental Credits

Verified cleanup data generates measurable environmental metrics. Potential for carbon credit integration. Blockchain-verified proof of impact.

Roadmap

Build, Test, Launch

Phase 1

Build Infrastructure

  • Database architecture for Trash Coins
  • API endpoints for task submission/validation
  • Photo verification system (GPS, timestamps)
  • Trash Bank Card system (NFC, QR)
Phase 2

Test Concept (McClaw)

  • Pilot with 10-50 users
  • McClaw task escrow integration
  • Manual validation by agents
  • Collect feedback, iterate
Phase 3

Modify Infrastructure

  • Improve based on pilot findings
  • Scale validation capacity
  • Add anti-gaming measures
Phase 4

Launch Program

  • Full McClaw integration
  • 500+ Trash Collectors
  • First vending machine pilot
  • Corporate outreach
Phase 5

Develop DAO

  • Garbage Cans NFT contract (soulbound)
  • Quadratic voting mechanism
  • Sub-community formation
  • Budget distribution system
Phase 6

Launch DAO

  • First Trash Master election (bi-weekly)
  • Weekly budget distribution
  • Community governance active
  • Evaluate city expansion
