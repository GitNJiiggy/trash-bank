# Trash Bank — Competitive Differentiation

## What Makes Us Different

### vs. Plastic Bank

| Plastic Bank | Trash Bank |
|--------------|------------|
| Centralized platform | Decentralized DAO governance |
| No token for trading | TTKN is tradeable on DEX |
| Corporate-focused (B2B) | Consumer-first, B2B second |
| No governance | Quadratic voting by contributors |
| No homeless inclusion | Vending machines for unbanked |

**Enhancement by McClaw:** McClaw provides instant on-chain escrow and verification infrastructure that would take Plastic Bank years to build.

### vs. ZeLoop

| ZeLoop | Trash Bank |
|--------|------------|
| Plastic bottles only | All trash types |
| Consumer-facing app | DAO infrastructure |
| No governance | Community DAO |
| No homeless pathway | Vending machine network |
| Points system | Tradeable tokens |

**Enhancement by McClaw:** ZeLoop validates manually. McClaw's agent verification system scales faster.

### vs. Plastiks

| Plastiks | Trash Bank |

| Credit resale marketplace | Direct redemption |
| No community governance | DAO by contributors |

**Enhancement by McClaw:** McClaw's Base integration means instant MCLAW rewards alongside Trash Bank's coins.

### vs. Empower

| Empower | Trash Bank |
|---------|------------|

| Collectors sell credits | Direct utility |

**Enhancement by McClaw:** McClaw provides the task escrow layer that Empower would need to build from scratch.
